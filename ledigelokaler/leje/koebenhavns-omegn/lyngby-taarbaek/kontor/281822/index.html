Beskrivelse

Flotte lyse kontorer i markant ejendom i det attraktive omra°de omkring DTU, og tæt pa° motorvejen udlejes nu.Pa° 2. sal findes flotte 1.517 m2 indrettet med 3 the-køkkener, flere mødelokaler og gode lyse kontorer opbygget i glas.Pa° 3. sal pa° Lundtoftega°rdsvej 95 findes velindrettede 1563 m2 kontor, indrettet som a°bne lyse lokaler med med enkelte kontorer/mødelokaler opbygget i glas. Støttefunktioner som møde, print og thekøkkener samlet i kernen af bygningen. Markant ejendom i det attraktive omra°de omkring DTU, og tæt pa° motorvejen.Lager/laboratorie med port mod terræn findes i stue-etagen.
