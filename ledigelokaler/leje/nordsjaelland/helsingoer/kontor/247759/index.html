Intro tekst

191 m² kontor-/klinik-/behandlinglokale

Flashline

Kontor-/kliniklejemål

Beskrivelse

I den hyggelig havneby, Helsingør, på Axeltorv med smukt udsyn til torvet med alle de dejlige caféer og liv, findes dette skønne kontor-/kliniklejemål som kan lejes alene eller som opdelt. Ejendommen har flere lejere som er sundhedsrelateret men lejemålet kan også bruges som kontor, undervisnings eller call center.
