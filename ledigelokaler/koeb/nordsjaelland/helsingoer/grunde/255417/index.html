Title: Egeskovvej 50 - 3490 Kvistgård

Intro tekst

Helsingørs sidst udstykkede og største erhvervsområde tilbydes nu til salg

Flashline

Fin erhvervsgrund til salg

Beskrivelse

På Helsingørs største og sidst udstykkede erhvervsområde, kan vi tilbyde denne erhvervsgrund - på samlet - ca. 36.000 m2, som kan opdeles i minimum 10.000 m2 pr. grund.
Bebyggelsesgraden er 40% og området er udlagt til erhvervsformål, såsom industri-, lager- og værkstedsvirksomhed.
Lokal - kommuneplan for området er at finde på Helsingør Kommunes Hjemmeside under:
Start redegørelse for Lokalplan 2.13 og kommune plan tillæg 31.


Der kan placeres butikker til særlig pladskrævende varegrupper op til 5.000 m2 foruden tilknyttet byggemarked op til 2.000 m2 . Erhervsgrunden er perfekt for etablering af boxbutikker eller lignende.

Prisen er 555,00 kr. pr. m2 + moms.
