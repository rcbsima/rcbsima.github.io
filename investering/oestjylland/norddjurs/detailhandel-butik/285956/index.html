Intro tekst

En ejendom med rigtig mange muligheder

Flashline

VELBELIGGENDE FORRETNINGSEJENDOM MED VÆRKSTED IØRSTED

Beskrivelse

I centrum af Østed finder du denne forretningsejendom, som for den rigtig køber rummer rigtig mange muligheder. 
 
Lokalerne vil bla. kunne bruges til
 

  • Forretning med butikVærksted
  • Hobby
  • Udstilling
  • Lager
  • Indretning og udlejning af depotrum 
  • Opdeling i flere separate lejemål


 
Ejendommen fremtræder såvel udvendig som indvendig i ok stand.
 
Ejendommen er beliggende på dobbeltgrund og er opvarmet med fjernvarme, varmepumpe og el, ligesom der er solceller på taget.
 
Ejendommen er indrettet med 
 
  • Stort butikslokale
  • KontorKøkken med spiseplads
  • Toilet
  • VærkstedLager uden port
  • Lager med port
  • Kontor på overetage


 
Ejendommen er beliggende på stor dobbelt grund med meget udenomsplads.
